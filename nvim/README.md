# nvim configurations

## Directory Structures

root ━ init.vim         # root config
     │ keymappings.vim  # key mappins
     │ plug.vim         # Vim-Plug config
     ┣ pluins/          # plugins configs
     ┣ plugged/         # after installed plugins
     ┕ colors/          # color themes

## Plugins
